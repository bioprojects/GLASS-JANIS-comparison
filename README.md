# GLASS-JANIS-comparison
Programs used for the paper "Comparison of de-duplication methods used by WHO Global Antimicrobial Resistance Surveillance System (GLASS) and Japan Nosocomial Infections Surveillance (JANIS) in the surveillance of antimicrobial resistance" by Toshiki Kajihara, Koji Yahara, Sergey Romualdovich Eremin, Barbara  Tornimbene, Visanu Thamlikitkul, John Stelling, Aki Hirabayashi, Eiko Anzai, Satoyo Wakai, Nobuaki Matsunaga, Kayoko Hayakawa, Norio Ohmagari, Motoyuki Sugai, and Keigo Shibayama.
